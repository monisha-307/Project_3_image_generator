# 🎨 AI Image Generator

## 📌 Project Overview
This project is an AI-powered Image Generator built using Python and Streamlit. Users can enter a text prompt, and the application generates an image based on the given description.

## 🚀 Features
- Generate images from text prompts
- Simple and user-friendly interface
- Built using Streamlit
- AI-powered image generation

## 🛠️ Technologies Used
- Python
- Streamlit
- Google Generative AI API

## 📂 Project Structure

```
Project_3_Image_Generator/
│── main.py
│── requirements.txt
│── README.md
```

## ▶️ How to Run the Project

1. Clone the repository

```bash
git clone <repository-link>
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run main.py
```

## 📷 Sample Output

Enter a text prompt such as:

```
A celestial whale swimming through the night sky above a city
```

The application generates an image based on the given prompt.
